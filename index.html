Introductions

Welcome to my portfolio. This is clearly a work in progress.

To keep it short and sweet, I'm a first-year master's degree student in Computer Science at the University of Tampere. I'm focusing on UI and UX design, with a great interest in accessibility. One might wonder why I would have a gnarly looking static website as a portfolio, and truthfully I don't think I have a good answer for you. Maybe this will show that I can do some HTML, CSS and JS programming if necessary. I do also like it, but it's just not my main skill.

Also, I want to mention it here as well as on the GitHub page; absolutely no generative AI is or will be used in the making of this portfolio! Since the point is to practice my skills, it would be counterproductive to have something code this for me. My choices may not be the most optimal and I'm sure there are some more efficient ways to write things, but this is just my way of doing this.

You can find my most used resources on the resources page. I will try to keep it updated as I add more cool things into this portfolio, but I make no promises.
