Resources

As promised, here is a dedicated page for most of the resources I use while creating this portfolio. I've taken one university course on Web Development, which of course isn't a lot. For that reason I spend a fair amount of time browsing the links below. Is there anything else to say, really?

  • W3Schools

    W3Schools has been a lifeline for me. Not only does it provide really easy-to-undestand definitions for concepts, it also shows you what things look like when properly implemented. It covers pretty much everything I need to know, not only for web development stuff but also for everything programming related. I've used it to learn C++, SQL, JavaScript, HTML, CSS... the list could genuinely go on. A wonderful resource.

  • Stack Overflow

    No surprises here for those who do any programming. Without incriminating myself too much, there is a possibility that my code doesn't always work as I would want it to work. That's when Stack Overflow saves me from having to ask the questions myself, since they usually are one of the over 20 million questions that have been asked on the website.

  • GitHub Community

    The GitHub community discussions have proven to be a great tool in troubleshooting anything regarding the GitHub Pages themselves. As can be seen from this portfolio's URL, it is hosted through GitHub's wonderful (and free) Pages function. From file paths to problems with deployments, my questions have always already been asked in the discussion forum. Once again, a wonderful resource to make creating easier.
